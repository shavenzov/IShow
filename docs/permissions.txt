Для ограничения функционала приложения IShow можно использовать одну из следующих команд.

allow  - перечисление доступного функционала, весь остальной функционал будет запрещен
denied - перечисление функционала который будет запрещен, весь остальной функционал будет разрешен

Можно использовать только одну команду. Либо allow, либо denied. Если эти параметры не используются, то все разрешено.

Примеры:

allow=EXPAND,REMOVE_NODE,REMOVE_EDGE,LAYOUT_SETTINGS,SEARCH,NAVIGATOR,NODE_PROPERTIES

Разрешено : раскрывать узлы, удалять связи, изменять параметры раскладки, функционал поиска, функционал навигатора, редактировать свойства узла. Всё остальное запрещено.

denied=EXPAND,REMOVE_NODE,REMOVE_EDGE,LAYOUT_SETTINGS,SEARCH,NAVIGATOR,NODE_PROPERTIES 

Запрещено : раскрывать узлы, удалять связи, изменять параметры раскладки, функционал поиска, функционал навигатора, редактировать свойства узла. Всё остальное разрешено.

Ниже приведен список возможных значений:           


/**
		 * Создать связь 
		 */		
		CREATE_EDGE
		
		/**
		 * Удалить связь 
		 */		
		REMOVE_EDGE
		
		/**
		 * Удалить узел 
		 */		
		REMOVE_NODE
		

/**
		 * Выбрать все узлы 
		 */		
		SELECT_ALL  
		
		/**
		 * Выбрать все узлы определенной сети 
		 */		
		SELECT_ALL_NET_NODES
		
		/**
		 * Выделить все связи связанные с объектом 
		 */		
		SELECT_ALL_RELATIONAL_EDGES 
		
		
		/**
		 * Сделать узел корневым 
		 */		
		SET_NODE_AS_ROOT
		
		
		
		/**
		 * Развернуть узел 
		 */		
		EXPAND    
		
		/**
		 * Отобразить информацию об объекте 
		 */		
		SHOW_INFO  


		
		
		
		/**
		 * Обновить 
		 */		
		REFRESH 
		
		/**
		 * Свойства узла 
		 */		
		NODE_PROPERTIES 
		
		/**
		 * Свойства связи 
		 */		
		EDGE_PROPERTIES 
		
		/**
		 * Сгрупировать выделенные объекты 
		 */		
		GROUP_SELECTED_NODES 
		
		/**
		 * Загрузить объект асоциированный с узлом 
		 */		
		DOWNLOAD 
		
		/**
		 * Открыть изображение 
		 */		
		OPEN_IMAGE 
		
		/**
		 * Привязать изображение графа
		 */		
		LINK_OBJECT_TO_GRAPH_IMAGE 
		
		
		/**
		 * Функционал сохранения графа в каком либо из форматов 
		 */		
		SAVE_GRAPH 
		
		
		/**
		 * Функционал отмены действий 
		 */		
		UNDO_AND_REDO 
		
		/**
		 * Функционал прокрутки "лапкой" 
		 */		
		SCROLL_MODE 
		
		/**
		 * Анимировать направление связей 
		 */		
		ANIMATE_DIRECTION_RELATIONS 
		
		/**
		 * Функционал навигатор 
		 */		
		 NAVIGATOR 
		
		/**
		 * Изменять раскладку 
		 */		
		CHANGE_LAYOUT 
		
		/**
		 * Функционал настройки раскладки 
		 */		
		LAYOUT_SETTINGS;
		
		
		
		/**
		 * Изменять масштаб 
		 */		
		 CHANGE_SCALE 
		
		/**
		 * Функционал поиска 
		 */		
		SEARCH 
		
		/**
		 * Функционал отображения информации о приложении 
		 */		
		APPLICATION_INFORMATION
